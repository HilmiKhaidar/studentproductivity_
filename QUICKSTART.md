## ğŸ¯ First Time Setup (5 Menit)

### Step 1: Set Target Tidur (30 detik)
1. Klik **Pengaturan** di sidebar
2. Set "Target Jam Tidur": **8 jam**
3. Set "Jam Tidur Ideal": **22:00**
4. Set "Jam Bangun Ideal": **06:00**
5. Klik **Simpan Pengaturan**

### Step 2: Buat Target Semester (1 menit)
1. Klik **Target** di sidebar
2. Klik **Tambah Target**
3. Isi:
   - Judul: "IPK Semester Ini 3.5"
   - Deskripsi: "Fokus di semua mata kuliah"
   - Kategori: "Akademik"
   - Target Tanggal: [Akhir semester]
4. Tambah Milestone:
   - "Nilai A untuk Kalkulus"
   - "Nilai A untuk Pemrograman"
   - "Nilai B+ untuk Fisika"
5. Klik **Tambah**

### Step 3: Input Tugas Minggu Ini (2 menit)
1. Klik **Tugas** di sidebar
2. Klik **Tambah Tugas**
3. Buat 3-5 tugas:

**Contoh Tugas 1:**
- Judul: "Tugas Kalkulus Bab 3"
- Deskripsi: "Soal nomor 1-20"
- Kategori: Assignment
- Prioritas: High
- Deadline: [3 hari dari sekarang]
- Estimasi: 120 menit

**Contoh Tugas 2:**
- Judul: "Belajar untuk Quiz Pemrograman"
- Deskripsi: "Review materi array dan function"
- Kategori: Exam
- Prioritas: Urgent
- Deadline: [Besok]
- Estimasi: 90 menit

### Step 4: Setup Kebiasaan (1 menit)
1. Klik **Kebiasaan** di sidebar
2. Klik **Tambah Kebiasaan**
3. Buat 2-3 kebiasaan:

**Kebiasaan 1:**
- Nama: "Olahraga Pagi"
- Deskripsi: "30 menit jogging atau workout"
- Frekuensi: Daily
- Warna: [Pilih warna favorit]

**Kebiasaan 2:**
- Nama: "Baca Buku 1 Jam"
- Deskripsi: "Baca buku non-fiksi atau textbook"
- Frekuensi: Daily
- Warna: [Pilih warna]

### Step 5: Catat Tidur Kemarin (30 detik)
1. Klik **Tidur** di sidebar
2. Klik **Catat Tidur**
3. Isi data tidur kemarin:
   - Tanggal: [Kemarin]
   - Jam Tidur: 22:30
   - Jam Bangun: 06:00
   - Kualitas: Good
   - Gangguan: 1
4. Klik **Simpan**

## ğŸ“ Workflow Harian

### Pagi (5 menit)
1. **Buka Dashboard**
   - Lihat skor produktivitas
   - Check tugas hari ini
   - Review target aktif

2. **Planning**
   - Pilih 3 tugas prioritas
   - Estimasi waktu total
   - Tentukan urutan pengerjaan

### Siang - Sore (Kerja)
1. **Mulai Pomodoro**
   - Klik **Pomodoro** di sidebar
   - Pilih tugas dari list
   - Klik **Mulai**
   - Fokus 25 menit
   - Istirahat 5 menit
   - Repeat!

2. **Update Progress**
   - Check tugas saat selesai
   - Update milestone goals
   - Mark kebiasaan selesai

### Malam (2 menit)
1. **Review Hari Ini**
   - Lihat Analytics
   - Check completion rate
   - Catat tidur hari ini

2. **Planning Besok**
   - Tambah tugas baru jika ada
   - Set prioritas
   - Prepare mental
